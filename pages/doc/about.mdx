# **UCcoin**

# **简介**

UCcoin是optimism生态上的专注链上衍生品服务的创新性金融服务平台

**摘要：**

日新月异的行业外部环境随之带来的是更多财富获取的机会。2020年以来DeFi创造的颠覆性玩法已成功吸引众多资本青睐，给行业带来新的方向和可能；2021-2022年我们也迎来了新一轮的NFT浪潮，其结合资产稀缺性、不可分割性，清晰所有权、资产合成能力及更多行业应用可拓展性，将带领大量传统的互联网用户进入加密货币领域。

UCcoin致力于在两大领域的探索并力求技术创新发展， 同时UCcoin将布局并诠释全新的综合赛道，迎接Web3时代的巨大风口。

**项目概要：**

UCcoin将倾心打造一个全方位去中心化泛NFT评级估值自治平台，允许万物皆可NFT化，并鼓励用户喜爱的NFT评分，量化认可度。通过现有AIGC的技术和点票机制的底层架构为行业打造一个强劲，有趣且理性的生态，并成为NFT x DeFi 赛道的先行者。

平台用户以参与共识值贡献的行为强调'群体智慧'。 当投票结束时，通过AI票选机制进行筛选，预测正确的用户可以从其质押权益（NFT）中获取收益，从而真正推动并实现由权益相关者定义NFT市场价值。

同时，UCcoin 旨在为全网NFT生态打造测评标准，建立AI点票共识机制。 资产的价值发现来源于市场的共识度，UCcoin点票协议/共识选票机制允许并促进NFT的价值发现，完善并普及NFT品质标准，为去中心化NFT提供估值平台。

- **去中心化分布式自治组织** ：旨在打造一个去中心化社区自治NFT生态闭环，平台实现DAO概念，代币持有人（用户）通过'众智'进行决策并获取透明度及利益最大化。

- **质押机制** ：旨在为NFT价值打造共识度，用户质押NFT贡献共识值，通过众评和AI点票机制结果最终拟定NFT价值共识机制。

- **DeFi** 流动性：旨在结合DeFi特性满足提升NFT流通性的相关需求，用户可通过流动性矿池获取挖矿奖励。

**合成资产聚合器** ：旨在提供及展示铸造一体化中心

承载合成资产的想象空间，引流资产丰富性，满足大众化需求。

**权益股份化** ：旨在碎片化资产，拓宽用户投资市场，丰富选择类型，降低参与难度。



# **共识选票机制**

**逻辑斯蒂方程**  **Logistic Equation**

逻辑斯蒂方式作为经典分类算法，最初应用在解释生态学的种群增长现象，后在信息传播及商品预测销售的应用场景中的使用较为常见。其特点根本上体现为遵循S型曲线增长，增长过程分为慢-快-慢三个阶段。

于信息传播而言，初期会经历较低曝光率及回应度，但通过一定时间的传播，社会上大部分人群都会接收到这一信息；对于商品的销售而言，虽然初始销售量及活跃度较小，但是通过销售手段能够达到时间段内迅速增长，最终达到销量趋近饱和。

UCcoin技术团队相信，大部分初创的NFT资产在认可度及曝光率方面是缺乏的。而平台提出的共识贡献机制以逻辑斯蒂方程原理建立，可以帮助NFT资产实现S型增长曲线。

在UCcoin平台，上架或质押的NFT资产自动进入阶段 **1-**  **初始上升阶段** ，获得初曝光量；平台的共识度选票能为NFT的传播速度及影响力加速，获得贡献值的NFT能够在短时间内获取大量曝光度，提升并建立市场共识价值，随即进入阶段 **2-快速增长阶段**。随着时间的推移，当该NFT在平台得到一段时间的共识增持，其共识价值也会趋近平稳并放缓增长速度，进入阶段 **3-上升稳定阶段** 。最终，该NFT获得稳定的市场参考性并视为进入阶段 **4-动态稳定阶段**。

随着时间的增长，产生的曲线斜率会逐渐降低
